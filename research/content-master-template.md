# CONTENT MASTER TEMPLATE & QA RULES
**Owner:** Topic 504 (Content Operations)
**Purpose:** Strict QA formatting rules for all Syndicate production lanes. If a script violates these rules, Topic 504 must kick it back to the writer.

## 1. THE SHORT-FORM GANG (VI / SA / PW)
**Duration:** Strictly under 45 seconds (approx. 110-130 words).
**Format:** Vertical (9:16).
**Script Rules:**
- NO STAGE DIRECTIONS in the final script block. (e.g., Do not include `*(music swells)*` or `[Camera pans]` in the text meant for the teleprompter/avatar).
- Must include Title, Description, and Platform Tags (YT Shorts, TikTok, IG Reels).
- Language: Italian for SA, English for VI, Multi (Thai/Korean/Vietnamese/Indonesian/etc) for PW.

## 2. THE LONG-FORM ENGINE (OceanFloor)
**Duration:** 4 minutes (approx. 600-700 words).
**Format:** Landscape (16:9).
**Structural Rules:**
- **The Hook (0:00 - 0:45):** The first 45 seconds must be a standalone narrative that ends on an open loop/cliffhanger driving traffic to `oceanfloor.sestito.com`. This chunk must be detachable for Shorts/TikTok.
- **The Deep Dive (0:45 - 3:30):** The core documentary segment.
- **The CTA (3:30 - 4:00):** Call to action directing to the website/AskStella funnel.
- **Visual Specs:** Must accommodate the "On-Location Reporter" format (Avatar full screen for hook, transitioning to corner for B-roll).
- NO STAGE DIRECTIONS in the spoken audio blocks.

## 3. QA CHECKLIST FOR TOPIC 504
Before clearing a script for the Captain to render, Topic 504 must verify:
- [ ] Is the word count appropriate for the format? (<130 words for short, <700 for long).
- [ ] Are all stage directions stripped from the teleprompter text?
- [ ] Are tags and descriptions included?
- [ ] (For OF only) Is the 45-second detachable hook clearly defined?

If the script fails any check, REJECT IT back to the writer.
