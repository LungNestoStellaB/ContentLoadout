## Purpose
VI, SA, PW scripts and shipping cadence. Nothing else.

## Daily Rhythm (This Lane)
- **Morning Shipping (00:00–06:00 UTC):** Deliver fitted scripts.
- **Rotation:** 5 days on / 5 days off. 2x VI, 2x SA, 2x PW per rotation.
- **Languages:** Thai, Korean, Vietnamese.

## Delivery Protocol
- PULL by default. Captain calls up packages when required.
- Push delivery only by explicit prior agreement.

## Content Request Format
When requesting content, provide:
- Script number (e.g., VI 22, SA 7, PW 9)
- Language (Thai, Korean, Vietnamese)
- Deadline if applicable

## Delivery Format
Every script delivered with:
- Title
- Thumbnail text options
- Hashtags
- Keywords
- CTA options

No partial drafts. Scripts must be complete and ready to ship.
